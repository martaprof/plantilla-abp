# Materiales para el alumnado

Esta carpeta reúne documentos que el alumnado usará directamente durante el proyecto.

## Materiales recomendados

- [`plantilla-planificacion-equipo.md`](plantilla-planificacion-equipo.md): organización de roles y tareas.
- [`diario-aprendizaje.md`](diario-aprendizaje.md): registro individual o grupal del proceso.
- [`plantilla-memoria-final.md`](plantilla-memoria-final.md): estructura para documentar el producto final.

## Cómo preparar estos materiales

Cada documento debe ser claro, breve y accionable. El alumnado debe saber qué completar, con qué nivel de detalle y cuándo entregarlo.
