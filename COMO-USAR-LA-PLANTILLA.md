# Cómo usar esta plantilla

Este repositorio está pensado para que un docente pueda descargarlo, renombrarlo y completar sus apartados hasta convertirlo en un proyecto ABP publicable en GitHub.

## 1. Primeros pasos

1. Cambia el nombre del repositorio por el título del proyecto.
2. Edita el [`README.md`](README.md) principal.
3. Completa los datos básicos: etapa, curso, materia, duración, producto final y autoría.
4. Define la pregunta guía.
5. Revisa [`TODOS.md`](TODOS.md) y elimina las tareas que no procedan.

## 2. Orden recomendado de trabajo

### Paso 1. Definir el proyecto

Completa:

- [`README.md`](README.md)
- [`01-contexto-y-justificacion/README.md`](01-contexto-y-justificacion/README.md)
- [`02-diseno-didactico/README.md`](02-diseno-didactico/README.md)

El objetivo de este paso es dejar claro qué se va a aprender, por qué es relevante y qué producto final realizará el alumnado.

### Paso 2. Relacionar el currículo

Completa:

- [`01-contexto-y-justificacion/relacion-curricular.md`](01-contexto-y-justificacion/relacion-curricular.md)

Incluye normativa, competencias específicas, criterios de evaluación, saberes básicos y evidencias.

### Paso 3. Diseñar las sesiones

Completa:

- [`03-sesiones/README.md`](03-sesiones/README.md)
- [`03-sesiones/sesion-01/README.md`](03-sesiones/sesion-01/README.md)
- [`03-sesiones/sesion-02/README.md`](03-sesiones/sesion-02/README.md)
- [`03-sesiones/sesion-03/README.md`](03-sesiones/sesion-03/README.md)

Para crear más sesiones, copia la carpeta [`03-sesiones/sesion-modelo`](03-sesiones/sesion-modelo/) y renómbrala.

### Paso 4. Preparar materiales

Completa:

- [`04-materiales-alumnado`](04-materiales-alumnado/)
- [`05-materiales-docente`](05-materiales-docente/)
- [`07-recursos-tecnicos`](07-recursos-tecnicos/)

Aquí se añaden plantillas, guías, enlaces, fichas, imágenes, esquemas, código o cualquier archivo necesario.

### Paso 5. Preparar la evaluación

Completa:

- [`06-evaluacion/README.md`](06-evaluacion/README.md)
- [`06-evaluacion/rubrica-general.md`](06-evaluacion/rubrica-general.md)
- [`06-evaluacion/lista-cotejo-proceso.md`](06-evaluacion/lista-cotejo-proceso.md)
- [`06-evaluacion/autoevaluacion.md`](06-evaluacion/autoevaluacion.md)
- [`06-evaluacion/coevaluacion.md`](06-evaluacion/coevaluacion.md)

La evaluación debe estar conectada con los criterios curriculares y con evidencias reales del proyecto.

### Paso 6. Documentar resultados

Cuando el proyecto se implemente, completa:

- [`08-resultados-y-evidencias`](08-resultados-y-evidencias/)
- [`09-mejoras-y-reutilizacion`](09-mejoras-y-reutilizacion/)

Guarda productos finales, capturas, fotografías, memorias, conclusiones y mejoras para futuras aplicaciones.

## 3. Convenciones de escritura

- Usa castellano correcto, con tildes, mayúsculas y signos de puntuación.
- Mantén los README breves pero completos.
- Usa tablas para datos estructurados.
- Usa diagramas Mermaid cuando ayuden a comprender procesos.
- Marca con `#TODO` lo que falte por completar.
- Elimina los `#TODO` cuando estén resueltos.
- Usa nombres de archivo claros y sin espacios.

## 4. Revisión antes de publicar

Antes de subir el repositorio a GitHub, comprueba:

- que el README principal explica bien el proyecto;
- que los enlaces internos funcionan;
- que no quedan `#TODO` importantes sin justificar;
- que las imágenes se ven correctamente;
- que los archivos técnicos están en la carpeta adecuada;
- que la evaluación está alineada con el currículo;
- que no se publican datos personales del alumnado sin autorización.

## 5. Recursos de apoyo sobre Git y GitHub Classroom

Para docentes que no hayan trabajado antes con repositorios, pueden consultarse estos recursos:

- [Aspectos básicos de Git](https://docs.github.com/es/get-started/git-basics), documentación de GitHub para configurar y usar Git.
- [Inicio rápido para GitHub](https://docs.github.com/es/get-started/start-your-journey/hello-world), guía inicial para crear y editar repositorios.
- [Acerca de GitHub Classroom](https://docs.github.com/es/education/manage-coursework-with-github-classroom/get-started-with-github-classroom/about-github-classroom), introducción a la herramienta para crear clases y tareas digitales.
- [Teach with GitHub Classroom](https://docs.github.com/en/education/manage-coursework-with-github-classroom/teach-with-github-classroom), guía para docentes sobre aulas, tareas y uso de plantillas.
