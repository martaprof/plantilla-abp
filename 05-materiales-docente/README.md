# Materiales para el docente

Esta carpeta contiene recursos que ayudan a preparar, guiar y evaluar el proyecto. Puede incluir solucionarios, notas de implementación, ejemplos resueltos o respuestas esperadas.

## Contenido recomendado

- [`orientaciones-docentes.md`](orientaciones-docentes.md): guía breve para aplicar el proyecto.
- [`solucionario-orientativo.md`](solucionario-orientativo.md): respuestas, ejemplos o criterios de corrección.
- [`registro-observacion.md`](registro-observacion.md): tabla para recoger evidencias durante el trabajo en aula.

## Uso

Estos materiales pueden mantenerse como apoyo interno del docente o publicarse si se desea que el repositorio sea completamente reutilizable.
