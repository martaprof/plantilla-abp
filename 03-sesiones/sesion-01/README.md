# Sesión 01: presentación del reto

## Objetivo

Presentar el proyecto, activar conocimientos previos y formular la pregunta guía.

## Desarrollo sugerido

1. Presentación de una situación real o cercana relacionada con el proyecto.
2. Lluvia de ideas sobre problemas, necesidades y posibles soluciones.
3. Formulación o revisión de la pregunta guía.
4. Presentación del producto final y de las fases de trabajo.
5. Organización inicial de equipos y roles.

## Evidencias

- Pregunta guía validada.
- Primer listado de ideas y dudas.
- Equipos formados.
- Roles iniciales asignados.
