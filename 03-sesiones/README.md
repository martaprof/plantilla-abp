# Sesiones

Esta carpeta organiza el desarrollo del proyecto sesión a sesión o por bloques de trabajo. Puedes crear tantas carpetas como necesites.

## Cómo nombrar las sesiones

Se recomienda usar este formato:

```text
sesion-01-presentacion-reto
sesion-02-investigacion-inicial
sesion-03-diseno-solucion
```

Si el proyecto se organiza por bloques, también puedes usar:

```text
bloque-01-investigacion
bloque-02-diseno
bloque-03-desarrollo
```

## Estructura recomendada de cada README de sesión

Cada sesión debería incluir:

- objetivo de la sesión;
- relación con la pregunta guía;
- saberes o contenidos;
- materiales;
- desarrollo paso a paso;
- producto o evidencia;
- orientaciones para el docente;
- tareas para casa, si procede;
- `#TODO` pendientes de enlaces, imágenes, capturas o archivos.

Puedes copiar [`sesion-modelo/README.md`](sesion-modelo/README.md) para crear nuevas sesiones.
